# Dog Walkings


### Project

#### Dependencies

- Docker 18.09
- Docker-compose 1.21

#### How to run

run:

```
./scripts/setup
```

then:
```
./scripts/dog_walkings
```

#### Running the tests

```
./scripts/dog_walkings rspec
```
